# 🎓 StudentGPT

🚀 **StudentGPT** is an AI-powered web application designed to assist students with academic tasks, learning, and productivity. It provides instant responses, explanations, and support across various subjects to enhance the learning experience.

🔗 Live Demo: https://student-gpt.onrender.com/

---

## 📌 Features

* 🤖 AI-powered question answering
* 📚 Helps with academic concepts and explanations
* ⚡ Fast and user-friendly interface
* 🌐 Accessible from any device with internet
* 🧠 Supports multiple subjects and topics
* 💬 Interactive chat-based UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Express,Node.js
* **AI Integration:**Gemini
* **Deployment:** Render



## 🎯 Use Cases

* Students preparing for exams
* Quick doubt solving
* Learning new concepts
* Assignment assistance
